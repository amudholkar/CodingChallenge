How to use the Program

View Tables: Select one of the restaurants to view all the tables with their corresponding waiters

View Assign Tables: Select of the waiters to view Tables assigned per restaurant 

Assign Table: Select a Restaurant - then Select the Table you want to Assign from the list of unassigned tables and the waiter form the list of available waiters - then click assign. After you click assign the table disappears and the if the the number of tables assigned to the waiter has reached 4 the waiter is removed from the list as well. 

To check if assignment is successful click the back to home button and view assigned tables -> waiter

You might have to switch to another waiter and then click back to the waiter you are interested in, to reflect changes in assignment
